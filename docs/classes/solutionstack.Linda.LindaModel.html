set()

set(\solutionstack\Linda\type  $data = array()) : $this

Update/set properties on the row model, this sets data on the database columns.<br/> The string NOW() and TIME() can be used to insert PHP date values, the formats used are <br/> <code> date("Y-m-d")</code> and <code> date("Y-m-d H:i:s")</code> respectively call #save when done to commit the data.

  
    the format of the argument, is an array with name-value pairs representing column names and values
    //eg
    $data = array( "age"=>2, update_timestamp=>"TIME()");
